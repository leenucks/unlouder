Working

easy explained, difficult to CODE.

Upload

U provide direct link, it downloads those files and save 'em to server (Local File upload soon). SHA256-checksum is generated for file for later retrieving and preventing duplicate upload. Current upload cap of single file is 2 GiB as of it uses Telegram Storage and no file-splitting is done (yet).

Checksum

Uploaded files return back with SHA256-checksum of file. All files are Private unless u got the file checksum but are accessible by owner/admin. For Private server, I may Open-source the project later, Go Host your own üòè

Download

Files are retrieved using SHA256-checksum from Telegram and streamed directly to u. (Download links never expire, but server can :v )
